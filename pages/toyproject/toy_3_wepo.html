WePo: 포트폴리오 공유 웹 서비스
Github

우리들의 포트폴리오, Wepo는 개발자들의 포트폴리오 공유를 목적으로 제작된 웹 서비스입니다.

사용기술

  • FE : TypeScript, React.js, Recoil, ReactQuery, ReactHookForm, Axios, styled-components
  • BE : JavaScript , Node.js, Express.js, MongoDB, Mongoose
SOHI's WORK IN PROJECT
백엔드
  • 자격증 정보, 학력 정보 CRUD 구현
  • multer를 활용한 유저 프로필 이미지 업로드 구현
  • MongoDB aggregation연산자를 활용하여 해당 유저의 정보들을 그룹화하여 조회 구현
  • MongoDB $inc operator를 활용한 유저ID 부여 및 유저 정보 조회 URL 간결화
  • 다른 유저의 포트폴리오 즐겨찾기 기능 구현
  • ERD, API 문서 작성
기획 / 디자인
  • 요구사항을 분석하여 기능명세서 작성 및 페이지 구조 설계
  • 초기 요구사항 이외의 개선사항, 추가기능 기획
  • 요구사항에 맞는 웹 및 UI 디자인 디자인 작업물 보기 (Figma)

로그인, 회원가입

로그인, 회원가입 UI는 간결하고 깔끔하게 구현하였습니다.

로그인, 회원가입 input은 입력 시 실시간으로 벨리데이션을 진행하여 사용자 경험을 개선하였습니다.

회원가입 시 이메일 인증 절차를 구현하였습니다.

비밀번호 5회 오입력 시 임시 비밀번호를 이메일로 전송하는 기능을 구현하였습니다.

기본 프로필 이미지는 랜덤으로 설정되도록 구현하였습니다.

포트폴리오 조회 및 검색

포트폴리오를 등록한 사용자들의 목록을 최신순으로 조회할 수 있습니다.

등록된 관심분야별로 목록을 필터링하거나 이름, 이메일 등으로 검색을 하여 원하는 포트폴리오를 조회할 수 있습니다.

포트폴리오 등록 및 수정

나의 포트폴리오 페이지에서 이름, 한줄 설명, 프로필 사진을 수정하고 비밀번호를 변경할 수 있습니다.

자격증, 수상 이력, 프로젝트, 학력 내용을 작성하거나 수정, 삭제할 수 있습니다.

입력한 내용의 순서를 드래그 앤 드롭으로 간편하게 수정할 수 있습니다.

즐겨찾기

다른 사용자의 포트폴리오를 즐겨찾기할 수 있습니다. 즐겨찾기 한 포트폴리오는 마이페이지에서 확인 가능합니다.
