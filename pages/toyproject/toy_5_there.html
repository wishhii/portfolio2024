THERE: 서울 핫플레이스 공유 서비스
Github

THERE은 매주 서울의 핫플레이스를 선정하여 사용자들에게 추천하는 매거진 형태의 서비스입니다.

매거진에 선정된 장소에 대해 사용자들이 별점, 리뷰를 남기는 등의 정보 공유가 가능합니다.

사용기술

  • JAVA, Spring, JSP, MySQL, JQuery, Bootstrap
  • Eclipse, Git, Aquery Tool
SOHI's WORK IN PROJECT

  • 플레이스 목록 및 상세 페이지 화면 구현

  • 마이페이지 화면 구현

관리자 페이지

관리자 페이지는 Bootstrap을 사용하여 깔끔한 UI를 효율적으로 구현하였습니다.

메인 페이지에서는 최근 등록된 플레이스, 회원, 매거진 게시물을 요약하여 확인할 수 있습니다.

각 메뉴에서는 table 형태로 정리된 게시물 목록을 확인하고, 게시물을 추가/수정/삭제할 수 있습니다.

플레이스를 등록할 때 카테고리, 이름, 주소, 썸네일 등을 입력할 수 있습니다.

대표 썸네일은 1개만 등록할 수 있습니다.

Kakao Map API에서 제공하는 주소좌표변환객체를 통해 관리자가 주소를 텍스트로 입력하면 해당 주소에 맞는 위치를 지도에 Marker로 표시합니다.

매거진 페이지

괸리자가 매주 1개씩 새로운 주제의 매거진을 발행하여 사용자들에게 플레이스 정보를 제공합니다.

매거진에서 추천한 플레이스들은 플레이스 목록에 추가되어 지역별/카테고리별 플레이스 페이지에서도 확인할 수 있습니다.

지역별 / 카테고리별 플레이스 추천 페이지

'카테고리별 추천 플레이스' 페이지의 경우 관리자가 등록한 플레이스가 카테고리별로 분류되어 사용자에게 제공됩니다.

상단 카테고리 아이콘을 클릭을 통해 탭 형식으로 구현하였습니다.

관리자가 플레이스 등록 시 '지역별 추천 플레이스' 페이지 구현에 필요한 정보만 별도의 json 파일로 만들어 저장하는 과정을 추가하고, 해당 페이지에서는 json을 전달받아 지도를 구현하였습니다.

지도의 마커를 클릭하면 플레이스에 대한 간단한 정보가 인포윈도우로 보여집니다.

지도 좌측의 목록 클릭하면 해당 플레이스의 마커로 지도의 중심이 이동하도록 구현하였습니다.

상단 지역구 버튼을 클릭하면 해당 지역구의 중심좌표로 지도가 이동하고 해당 지역구 내의 등록된 플레이스들의 목록과 마커로 업데이트됩니다.

내 주변 플레이스 보기 버튼을 클릭하여 내 위치를 기반으로 플레이스 정보를 탐색할 수 있습니다.

플레이스 상세 페이지

플레이스에 대한 상세한 정보와 좋아요, 리뷰 등 사용들의 작성한 정보를 확인할 수 있습니다.

로그인한 사용자에 한해 좋아요, 리뷰쓰기 기능을 이용할 수 있습니다.

리뷰는 모달을 통해 화면 이동 없이 작성할 수 있습니다.

마이페이지

마이페이지를 통해 개인정보를 수정할 수 있습니다.

좋아요를 누른 플레이스와 매거진 목록을 마이페이지 하단에서 확인할 수 있습니다.

내가 작성한 모든 리뷰를 확인하고 수정/삭제할 수 있습니다.
