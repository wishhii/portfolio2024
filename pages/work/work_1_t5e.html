배수펌프장 3D 통합 모니터링 UI 개발
TypeScript Django

배수펌프장 3D 통합 모니터링 시스템은 배수펌프장과 그 일대의 장비/설비의 실시간 상태를 연동하고 3D로 가시화하여 모니터링하고 제어할 수 있도록 하며,

위험상황/재난상황 등이 발생 시 실시간으로 가시화하여 보여주는 디지털트윈 3D 재난 관제 모니터링 서비스입니다.

개발팀의 인턴 프론트엔드 개발자로 참여하여 해당 프로젝트에서 통합 모니터링 시스템의 대시보드 UI를 구현하였습니다.

통합 모니터링 시스템을 통해 실시간 모니터링뿐만 아니라 펌프장 내 장비와 설비를 실제로 동작시킬 수 있으며 재난 상황에 대한 알림과 대응이 중요한 만큼

직관적인 UI/UX를 구현하는 데 중점을 두었습니다. 또한, 최대한 모든 상황을 테스트하여 시스템 내 오류를 최소화하는 것을 목표로 작업하였습니다.

각종 장비/설비 데이터를 받아와 실시간 상태를 효율적으로 시각화하고 장비를 직접 제어하기 위한 컨트롤러 UI 컴포넌트를 구현하였습니다.

javascript의 class 문법을 사용해 UI를 컴포넌트 단위로 개발함으로써 재사용성을 높이고 유지 보수에 용이한 코드를 작성하였습니다.

배수펌프장에서 발생할 수 있는 여러 시나리오에 따라 요구되는 여러 UI들의 유기적이고 복잡한 동작 로직을 구현하였습니다.

예를 들어, 특정 알림이 오면 사이렌 소리가 재생되고 알람 팝업이 발생하며 관련 장비 컨트롤러 UI가 있는 대시보드 탭으로 이동하고 관련 구역의 CCTV 화면을 띄우고 3D 카메라 시점을 이동합니다.

또한, 실시간에 동시다발적으로 발생하는 알람의 순차적인 처리와 알람 관리를 위한 queue 형태의 알람 대기열 로직을 개발하였습니다.

ApexCharts.js를 사용해 배수펌프장의 장비별 로그 이력을 그래프로 구현하였습니다.

로그 이력은 장비, 기간 등으로 검색하여 조회가 가능하도록 구현하였고 검색한 조건에 맞춰 그래프가 업데이트 되도록 구현하였습니다.
